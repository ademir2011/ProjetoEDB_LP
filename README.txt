1. Crie a conta no http://www.github.com

2. Faça o download do Git http://git-scm.com/downloads
	2.1. 	Quando baixado instale, (next... next.. next..)
	2.2. 	O atalho do arquivo executável não fica na área de trabalho portanto vá em INICIAR procure por “git bash” e arreste o ícone para um local facilmente acessível

3. Para quem não deu git clone (enviou a pasta do projeto para o PC) ou excluiu a pasta do projeto no computador faça os seguintes passos:
	3.1. Abra o git bash...
	3.2. Inicialmente o git bash fica direcionado para o “meu computador”


	Portando navague até um local que você queira coloca a pasta do projeto. Para acessar uma pasta digite cd “local do pasta” e para voltar  cd ..
recomendo armazenar no disco local C		
	cd c:/
	
	

	3.3. Agora digite git clone “endereço do github”
		(esse endereço se encontra no site do github e serve para reconhecer qual repositório você quer baixar no seu computador \/ \/ \/ )

	

4. Depois de ter colocado o projeto no seu computador digite o comando git pull
que serve para atualizar o seu repositório com o repositório principal (nuvem)

IMPORTANTE: sempre que você for programar dê um git pull !!! Pois outros membros do projeto podem ter feito alguma alteração, e só vai esta no seu projeto se você atualizar !

5. Depois de programar e ter feito todas as alterações você precisa enviar para a nuvem, para isso digite os comandos abaixo

git pull				- (novamente) Atualiza seu repositório
git add -–all			- Pegando todos os arquivos		
git commit -m "mensagem" 	- Preparando para commitar e adicionando uma 							mensagem do que você esta commitando
git push origin master		- Enviando arquivo para o repositório principal

EXTREMAMENTE IMPORTANTE: sempre antes de programar dê git pull !! 

EXTREMAMENTE IMPORTANTE 2: sempre que for seguir o passado 5 dps de programar ou o passo 4 para atualizar não esqueça de navegar até o projeto no git bash
