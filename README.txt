Galera, para commitar (enviar um código para o repositório principal) é só digitar esses código no git bush

git pull				- Atualiza seu repositório
git add –all			- Pegando todos os arquivos		
git commit -m "mensagem" 	- Preparando para commitar e adicionando uma mensagem do que você esta 				commitando
git push origin master		- Enviando arquivo para o repositório principal
